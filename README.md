## üí° About This Project

Preparing for the HEC USAT can be challenging, especially when it's hard to find high-quality, organized past papers. This repository was created to solve that problem.

The goal is to be a central, easy-to-navigate collection of all available USAT past papers to help you prepare for your university admissions. This is a community-driven project, meaning it grows and gets better with contributions from students like you.

## üóÇÔ∏è How This Repository is Organized

All materials are organized into folders. You can browse the folders above and download any PDF you need. No special software is required.

The structure is based on the USAT test categories:

* **/USAT-Engineering (USAT-E):** Contains past papers for students applying to Engineering programs.
* **/USAT-Medical (USAT-M):** Contains past papers for Pre-Medical students.
* **/USAT-Arts-Humanities (USAT-A):** For students in Arts & Humanities.
* **/USAT-Computer-Science (USAT-CS):** For Computer Science applicants.
* **/USAT-General-Science (USAT-GS):** For General Science students.
* **/USAT-Commerce (USAT-Com):** For Commerce students.
* **/Shared-Materials:** Contains subject-specific guides (e.g., Verbal Reasoning tips, Quantitative Reasoning formulas) that apply to all test types.

## ü§ù How to Contribute

This project thrives on community contributions. If you have a past paper, study guide, or answer key that isn't in this repository, please help us add it!

There are two easy ways to contribute:

**Option 1: The Easy Way (Open an Issue)**
1.  Go to the **[Issues](To-Your-Repo/issues)** tab (replace this link with yours).
2.  Click **"New Issue"**.
3.  Give it a title like "Missing Paper: USAT-E 2024".
4.  In the description, **attach the file(s)** or provide a link where you found them.
5.  I will review it and add it to the repository.

**Option 2: The GitHub Way (Submit a Pull Request)**
1.  **Fork** this repository.
2.  Create a new branch (e.g., `git checkout -b add-paper-2023`).
3.  Add your files to the correct folder.
4.  Commit your changes (`git commit -m "Add USAT-M 2023 paper"`).
5.  Push to the branch (`git push origin add-paper-2023`).
6.  Open a **Pull Request**.
